# 📚 ことばずかん

小学生向け **ことば検索・漢字変換** Webアプリです。  
iPhone の Safari からそのまま使えます。**APIなし・サーバーなし**の完全静的サイトです。

---

## ✨ 機能

### 🔍 タブ1 「ことばをしらべる」
- 漢字・ひらがな・英語でキーワード検索
- よみがな・英語訳・意味・100文字説明を表示
- 入力中にリアルタイムでサジェスト
- 算数・国語・理科・社会の人気ワードをタップ一発検索
- 結果をクリップボードにコピー

### ✏️ タブ2 「かんじにする」
- ひらがなの問題文 → 教科書らしい漢字まじり文に変換
- 「ふりがなをつける」ON で読み仮名付き表示
- 長い語を優先して変換（誤変換を防止）
- 変換結果をコピー可能

## 📁 ファイル構成

```
/
├── index.html           ← メインページ（エントリーポイント）
├── manifest.json        ← PWA設定（ホーム画面追加対応）
├── .nojekyll            ← GitHub Pages Jekyll無効化（必須！）
├── css/
│   └── style.css        ← ポップなデザイン（子ども向け）
├── js/
│   └── app.js           ← アプリロジック（検索・変換）
└── data/
    ├── dictionary.js    ← ことば辞書（50語以上）
    └── convert-map.js  ← ひらがな→漢字変換マップ（500語超）
```

## 🚀 GitHub Pages で公開する手順（初心者向け）

### ─── ステップ 0：GitHubのアカウントを作る ───

1. ブラウザで **https://github.com** を開く
2. 右上の **「Sign up」** をクリック
3. メールアドレス・パスワード・ユーザー名を入力して登録する
4. メールで届いた確認コードを入力する
5. ログインできたら準備完了！

> 💡 すでにアカウントがある人はステップ1から始めてください。

---

### ─── ステップ 1：新しいリポジトリを作る ───

1. GitHubにログインした状態で、右上の **「＋」マーク** をクリック
2. **「New repository」** を選ぶ

   ```
   ┌─────────────────────────────────┐
   │ + ▼                             │
   │   New repository   ← ここ！    │
   │   New organization              │
   └─────────────────────────────────┘
   ```

3. 次の項目を入力する

   | 項目 | 入力内容 |
   |------|---------|
   | **Repository name** | `kotobazukan`（好きな名前でOK） |
   | **Description** | `小学生向けことば検索アプリ`（省略可） |
   | **Public / Private** | **Public** を選ぶ（無料アカウントの場合） |

4. **「Create repository」** ボタンをクリック

> ✅ リポジトリが作られると、空のページが表示されます。

---

### ─── ステップ 2：ファイルをアップロードする ───

1. リポジトリのトップページに **「uploading an existing file」** というリンクがある

   ```
   …or upload a file
   uploading an existing file  ← ここをクリック！
   ```

   ※ 見当たらない場合は **「Add file」→「Upload files」** でも同じ

2. ファイルのアップロード画面が開く

3. このアプリのファイルを **全部まとめてドラッグ＆ドロップ** する

   **アップロードするファイル一覧：**
   ```
   index.html
   manifest.json
   .nojekyll          ← 忘れずに！（ドットで始まるファイル）
   css/style.css
   js/app.js
   data/dictionary.js
   data/convert-map.js
   ```

   > ⚠️ `.nojekyll` はドットで始まる「隠しファイル」です。  
   > Mac の場合は `Cmd + Shift + .` で表示できます。  
   > Windows の場合は「隠しファイルを表示」を設定してください。

4. 画面下の **「Commit changes」** ボタンをクリック

   ```
   Commit changes
   ─────────────────────────────────
   Add files via upload  ← デフォルトのメッセージでOK
   [Commit changes]      ← このボタンをクリック！
   ```

5. アップロード完了！リポジトリにファイルが表示される

---

### ─── ステップ 3：GitHub Pages を有効にする ───

1. リポジトリのページ上部にある **「Settings」タブ** をクリック

   ```
   Code  Issues  Pull requests  Actions  Projects  Wiki  Security  Settings
                                                                   ↑ここ！
   ```

2. 左側のメニューから **「Pages」** をクリック

   ```
   📋 General
   🔐 Access
   🛡️ Code and automation
      └── Pages  ← ここ！
   ```

3. **「Source」** の設定を変更する

   ```
   Source
   ────────────────────────
   Deploy from a branch   ← これを選ぶ

   Branch
   ────────────────────────
   main    /  (root)      ← この設定にする
   [Save]                 ← 保存ボタンをクリック！
   ```

4. **「Save」** をクリック

---

### ─── ステップ 4：公開URLを確認する ───

1. ステップ3の **「Pages」** ページをリロード（再読み込み）する

2. しばらく（1〜10分）すると、次のような緑のバナーが表示される

   ```
   ✅ Your site is live at
   https://（あなたのユーザー名）.github.io/kotobazukan/
   ```

3. 「Visit site」ボタンをクリックすると、アプリが開く！

> ⏳ 最大で10分ほどかかることがあります。  
> 表示されない場合は少し待ってからリロードしてみてください。

---

### ─── ステップ 5：iPhone の Safari でホーム画面に追加する ───

1. Safari で公開 URL（`https://〜.github.io/kotobazukan/`）を開く

2. 画面下の **「共有」ボタン（□↑）** をタップ

3. **「ホーム画面に追加」** をタップ

4. **「追加」** をタップ

5. ホーム画面にアイコンが追加されて、アプリのように使えます！

## ⚠️ よくある質問

### Q: `.nojekyll` ファイルが見えない
**A:** ドットで始まるファイルは隠しファイルです。  
- Mac：`Cmd + Shift + .` で表示ON/OFFできます  
- Windows：エクスプローラーの「表示」→「隠しファイル」にチェック

### Q: アップロードしたのにサイトが表示されない
**A:** Pages の有効化後、最大10分かかる場合があります。  
ブラウザのキャッシュをクリア（`Cmd+Shift+R` / `Ctrl+Shift+R`）してみてください。

### Q: ファイルを更新したい
**A:** GitHubのリポジトリページで「Add file」→「Upload files」から  
新しいファイルをアップロードすると自動で更新されます。

### Q: 無料で使えますか？
**A:** はい。GitHubの無料アカウントで、**Public リポジトリ**なら  
GitHub Pages は無料で使えます。
